Engenheira de software focada na construção de produtos web, mobile e integrações sob medida.
Atuo do design à entrega — análise de requisitos, arquitetura, desenvolvimento, testes, deploy e manutenção contínua.

Aprendo por conta própria desde sempre. Hoje meu interesse mais forte está em **automações conversacionais** e **agentes baseados em LLMs**, mas trabalho confortavelmente em todo o ciclo de desenvolvimento full stack.

## Projeto em destaque

### Hermes & Éter Bot

Stack pessoal de assistentes integrada ao meu fluxo diário:

- **Bot Telegram** com briefings agendados, slash commands, transcrição de voz e processamento de imagens
- **Assistente desktop por voz** ativado por wake word ("Oi Hermes")
- **Integrações** Claude API · MCP-over-SSE · Google Calendar · Whisper local · Edge TTS
- **Arquitetura** containerizada (Docker), single-tenant, totalmente local

`TypeScript` · `Python` · `Docker` · `Anthropic Claude` · `MCP` · `faster-whisper` · `Vosk`
